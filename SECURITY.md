# Security

## Supported Versions

Security fixes target the latest release.

## Reporting

Open a private security advisory on GitHub or contact the repository owner.

## Data Handling

This plugin is designed for local execution. Tenant data can include sensitive personal information. Keep `tenants/`, `outputs/`, `generated/`, and `applications/` out of source control.

Never commit:

- Government IDs or immigration documents
- Street addresses
- Salary history
- References
- Real application logs
- Real resumes unless intentionally publishing a sample with all consent required

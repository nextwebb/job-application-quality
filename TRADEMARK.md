# Trademark Policy

This policy explains use of the "Job Application Quality" name.

## Code vs Name

The source code is licensed under the [MIT License](LICENSE). The project name, repository identity, and any future logos or visual marks are separate from the source-code license.

## Allowed Uses

You may use the name without asking when you are:

- referring to this project accurately
- saying your tool works with Job Application Quality
- describing a fork or derivative with attribution
- writing tutorials, reviews, research, or educational material
- contributing in GitHub issues, pull requests, and discussions

## Uses That Need Permission

Ask before using the name in a way that suggests official ownership, endorsement, or commercial affiliation, including:

- hosted services named "Job Application Quality"
- paid products using the project name as the product name
- domains, app-store listings, or social handles that imply official status
- "certified", "official", "verified", or "powered by" claims
- commercial use of future logos or visual identity assets

## Forks

Forks are welcome under MIT. Give commercial forks their own product name and describe lineage clearly, for example: "based on Job Application Quality."

## Contact

Open an issue in this repository for trademark questions.

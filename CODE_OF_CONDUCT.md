# Code of Conduct

## Our Pledge

We want Job Application Quality to be a practical, respectful project for people building safer job-search tooling. Participation should be harassment-free for contributors and users regardless of background, identity, experience level, location, or employment status.

## Standards

Helpful behavior includes:

- using respectful, inclusive language
- giving technical feedback with evidence and care
- accepting constructive criticism
- respecting maintainers' time
- protecting candidate privacy
- focusing on truthful, safe, human-reviewed workflows

Unacceptable behavior includes:

- harassment, threats, insults, or personal attacks
- publishing private information without permission
- pressuring maintainers for free labor or emergency support
- contributing features that enable spam, deception, or unsafe automation
- any conduct that would be inappropriate in a professional setting

## Enforcement

Maintainers may remove, edit, or reject issues, comments, commits, code, documentation, and other contributions that violate this Code of Conduct.

Enforcement actions may include:

1. Private warning with clear remediation expectations.
2. Temporary restriction from project spaces.
3. Permanent removal from project participation.

## Scope

This Code of Conduct applies in GitHub issues, pull requests, discussions, documentation, and any public space where someone represents the project.

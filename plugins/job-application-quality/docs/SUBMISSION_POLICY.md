# Submission Policy

## Browser-Assisted Applications

Allowed:

- Open official posting
- Read fields
- Draft answers
- Upload approved artifacts
- Pause before final submit

Requires explicit approval:

- Final submit
- Legal attestations
- Sensitive demographic questions
- CAPTCHA solving
- Account creation

Blocked:

- CAPTCHA bypass
- Unsupported work authorization statements
- Government ID or passport upload without explicit review
- Reference submission without explicit review

## Email

The default email path creates drafts or packets. Sending requires explicit approval that names:

- Recipient
- Subject
- Body version
- Attachments
